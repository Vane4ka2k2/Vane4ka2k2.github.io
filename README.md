# Vane4ka2k2.github.io (Персональный сайт-портфолио)

Исходный код персонального веб-сайта-визитки Ивана Минаева — разработчика C++, системного программиста и специалиста по цифровой обработке сигналов. 

Сайт доступен на GitHub Pages по адресу: [vane4ka2k2.github.io](https://vane4ka2k2.github.io)

## ✨ Особенности и дизайн
- **Современный премиум-дизайн**: Использование эффектов стекломорфизма (glassmorphism), градиентов, плавных теней и современных шрифтов (Inter, Outfit).
- **Интерактивный интерактивный фон**: Динамический анимированный фон на Canvas API, реагирующий на движение курсора мыши пользователя.
- **Двухцветная тема**: Полноценная поддержка Темной (по умолчанию) и Светлой тем с плавным переключением без перезагрузки страницы (с использованием CSS-переменных и JavaScript).
- **Адаптивная верстка**: Сайт полностью оптимизирован под мобильные устройства, планшеты и экраны высокого разрешения. Имеется удобное выдвижное мобильное меню.
- **Оптимизация (SEO)**: Внедрены мета-теги описания, ключевых слов, авторов и корректная иерархия заголовков для лучшего ранжирования поисковыми системами.

## 🛠️ Стек технологий
- **Структура**: HTML5 (семантическая верстка).
- **Стилизация**: Ванильный CSS3 (CSS Custom Properties, Flexbox, Grid, CSS Transitions, Media Queries).
- **Логика и анимация**: Ванильный JavaScript (ES6+, Canvas 2D Context API для фоновых эффектов, обработчики событий DOM).

## 📂 Структура проекта
- `index.html` — основная разметка страницы портфолио.
- `style.css` — стили сайта, медиа-запросы, описание тем (светлая/темная) и анимаций.
- `script.js` — интерактивная логика переключения тем, мобильного меню и код рендеринга частиц на Canvas.
- `CNAME` — конфигурация кастомного доменного имени для GitHub Pages.

## 🚀 Как запустить локально

Для запуска сайта не требуется веб-сервер или сборка. Просто откройте файл `index.html` в любом современном браузере.

Или используйте расширение **Live Server** в VS Code (или аналогичные плагины других IDE) для автоматического обновления страницы при изменении кода.
