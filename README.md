## 🌟 Особенности сайта

- 🎨 **Премиальный дизайн** — Тёмная и светлая темы с плавным переключением.
- 🌊 **Интерактивный Canvas-фон** — Анимированные синусоидальные звуковые волны и соединенные частицы.
- 📱 **Полный адаптивизм** — Оптимизировано для мобильных устройств, планшетов и широких экранов.
- 🗂 **Динамический фильтр проектов** — Удобная сортировка C++ библиотек, математических модулей и приложений.

---

## 💻 Технологический стек

- **HTML5 & Vanilla CSS** — Семантическая разметка и переменные темы.
- **Vanilla JavaScript (ES6+)** — Мобильное меню, ScrollSpy, динамический Canvas API.
- **FormSubmit API** — Отправка сообщений из формы обратной связи.

## 🚀 Локальный запуск

Для просмотра сайта локально достаточно открыть `index.html` в любом браузере:

```bash
git clone https://github.com/Vane4ka2k2/Vane4ka2k2.github.io.git
cd Vane4ka2k2.github.io
# Откройте index.html
```
